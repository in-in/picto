# careex

A starter-kit for [mjml](https://mjml.io/documentation/)   
Powered by [pug](https://pugjs.org/api/getting-started.html) and gulp

### Get your copy

```bash
git clone git@github.com:in-in/careex.git project && cd project
```

### Run and watch

```bash
npm start
```

### Build project

```bash
npm run dist
```
